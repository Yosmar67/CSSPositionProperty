Yosmar's exple. Using position: absolute;

An element with position: absolute; is positioned relative to the nearest positioned ancestor (an ancestor with a position other than static). If there is no such ancestor, it is positioned relative to the initial containing block (usually the viewport).

This div element is positioned using absolute positioning. It is placed 50 pixels from the left and 100 pixels from the top of its nearest positioned ancestor.
This nested div element is positioned using relative positioning. It is moved 20 pixels to the right and 10 pixels down from its normal position within the absolute positioned div.
